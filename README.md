# DONNEES1

**Enterprise Database Schema**

This SQL script creates tables and data for an enterprise database. To run it, follow these steps:

1. Open a terminal and install PostgreSQL if you haven't already.
2. Launch the command `psql` to access the PostgreSQL command line.
3. Enter the command `\c enterprise` to connect to the "enterprise" database.
4. Run the script SQL by typing `\i /path/to/your/script.sql`, replacing "/path/to/your/script.sql" with the absolute path to your SQL file.
5. Press Enter to execute each instruction in the script.
